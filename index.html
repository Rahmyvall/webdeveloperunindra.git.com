Career Journey I am a professional

Untuk hal ini, saya termasuk yang cukup beruntung. Saya memulai karir saya sebagai web developer pada saat saya masih berada pada semester 2 di perkuliahan Universitas Indraprasta PGRI. Hal ini berawal dari tawaran seorang dosen yang sedang mengajar salah satu topik mata kuliah pemrograman yang pada saat itu saya sangat aktif dalam menjawab segala pertanyaan yang diberikan pada awal-awal waktu pertemuan.Dosen tersebut memanggil saya tepat setelah sesi waktu mengajar selesai dan berkata: Ram, apa kamu menguasai pemrograman berparadigma Object Oriented Programming?,saya jawab saja dengan optimis, bisa. Walaupun, sebenernya saya juga sedang mendalaminya dan pada saat itu saya mungkin memang lebih awal mencari tahu tentang paradigma pemrograman lebih awal daripada teman-teman seangkatan.Dari sana, berawal karir pertama saya sebagai junior web developer dengan framework pertama saya yang menggunakan bahasa pemrograman PHP. Di fase ini, saya mendapatkan banyak sekali pengetahuan dan pengalaman, karena projek yang kami kembangkan berskala besar, sistem informasi akademik untuk universitas dan sekolah.Teknik-teknik modularisasi kode, optimalisasi kueri, versioning, dan banyak lainnya saya dapatkan semasa bekerjasama,menjadijunior web developer dengan dosen saya tersebut, walaupun sebenarnya gaji yang saya terima pada saat itu lumayan jauh dari UMR.Tetapi, saya sangat bersyukur karena bisa menimba ilmu-ilmu yang mungkin luput saya dapatkan jika hanya belajar saja tanpa ada kasus nyata seperti ini.Selepas dari itu, saya melakukan personal branding ala kadarnya untuk memperkuat citra saya sebagai developer seperti membuat script sederhana yang bermanfaat.

Php Development

Pada awalnya PHP merupakan kependekan dari Personal Home Page (Situs personal). PHP pertama kali dibuat oleh Rasmus Lerdorf pada tahun 1995. Pada waktu itu PHP masih bernama Form Interpreted (FI), yang wujudnya berupa sekumpulan skrip yang digunakan untuk mengolah data formulir dari web. Selanjutnya Rasmus merilis kode sumber tersebut untuk umum dan menamakannya PHP/FI.
